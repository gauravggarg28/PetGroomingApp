# Images Directory

Please add the following images to this directory:

## Required Images

1. **logo.png** - The BhaoBhao logo image
   - Should be a PNG file with transparent background
   - Recommended size: 200x100px or similar aspect ratio
   - Place in: `/client/public/images/logo.png`

2. **banner-grooming.jpg** - Banner image for the home page
   - Should show a man grooming a labrador
   - Recommended size: 1920x1080px or similar (16:9 aspect ratio)
   - High quality image for best display
   - Place in: `/client/public/images/banner-grooming.jpg`

## Image Usage

- The logo will be displayed in the navbar
- The banner image will be used as the hero section background on the home page
- Both images are referenced in the code and will automatically load once placed in this directory
